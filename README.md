# UnrealPak-Context-Menu
Add the compression function of UnrealPak to the context menu
##
- 将“UnrealPak-ContextMenu-With-Compression.bat”放至任意位置（纯英文路径）
- 在“Add UnrealPak To Context Menu.reg”中末行写入您的‘UnrealPak-ContextMenu-With-Compression.bat’路径（每个子文件夹以及程序后都要额外加一个‘\’）
- - 例如：“D:\\Work\\UnrealPak\\UnrealPak-ContextMenu-With-Compression.bat\”
- - 可看：“添加UnrealPak上下文菜单.gif”
- 运行“Add UnrealPak To Context Menu.reg”即可创建上下文菜单
